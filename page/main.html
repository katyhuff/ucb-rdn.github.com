How To Edit This Website¶

First steps¶

  1. Get an academic GitHub account
  2. Ask an administrator of the UCB-RDN group to be added to it.
  3. Press the Fork button on the top right of the repository
  4. If your username is username clone the repository at https://github.com/username/ucb-rdn.github.com .
  5. Follow the instructions in the README file to edit the site.

Things you’ll need for this¶

  1. A Python distribution that includes sphinx. I recommend anaconda .
  2. The ability to use make on the command line.
  3. The ability to use git on the command line.

Some resources¶

  • Sphinx-flavored restructured text .
  • A tutorial on git .
  • For now, this site is maintained by Katy Huff who will be happy to help you if you email her .
